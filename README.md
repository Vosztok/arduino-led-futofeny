Arduino led futófény

Projekt leiras:
Ez a projekt az Arduino unoval történő futófény megvalósítása.

Működés:
- több LED-et vezérel,
- oda-vissza mozgást valósít meg,
- gomb vezérléssel szabályozható működés

Felhasznált eszközök:
- 9 db LED,
- 9 db 220 ohm os ellenállás,
- Arduino Uno,
- 1 db nyomógomb,
- kábelek
